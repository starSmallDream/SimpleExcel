# SimpleExcel
方便操作Excel,可同时对多Sheet进行操作,多个Excel对象之前的操作是隔离的

> 期望此工具是Excel工具中最好用的

# 支持的功能点
* 基于Excel文件，复制单元格行
* 多Sheet写入Excel，可根据需要自定义实现单元格对象；支持自定义单元格的样式信息(可以根据数值进行不同层度上的配色)
* 多Sheet读取Excel，基于列索引读取数据并填充列表,支持合并单元格读取值
* 基于列值自动合并列级的单元格，需要配置
* 基于Class实现功能点,方便自定义实现化
